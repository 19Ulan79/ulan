Изучить SQL запросы - CREATE, INSERT, SELECT, UPDATE, DROP
так же создать файл docs.txt и написать что делает каждый из этих методов

1 CREATE - Служит для создания объектов базы данных!
2 INSERT - Оператор языка SQL,который позваляет добавить строки в таблицу,заполняя их значениями!Значения можно вставлять перечислением с помощью слово values и перечислив их в круглых скобках через запятую или оператором select!
3 SELECT - Возвращающий набор данных из базы данных!
4 UPDATE - Обновляет словарь элементами из другого объекта словаря или из итерируемых пар ключ - значения!
5 DROP - Удаляет все таблицы в базе данных и удаляет саму базу данных!
